# Moral Latitude
### Navigating ethics through inquiry, complexity, and open minds

A free, open pedagogical resource for college and university ethics instruction, available at **[morallatitude.org](https://morallatitude.org)**.

---

## What This Is

Moral Latitude is a teaching resource built on a simple conviction: ethical dilemmas are valuable not for the answers they produce, but for the questions they force us to sit with. Most online ethics resources treat moral reasoning as a puzzle with a correct solution. This one doesn't.

The site offers two kinds of content:

- **Ethical Frameworks** — pages devoted to specific moral theories, using purpose-built cases to illuminate the internal logic of each tradition, and a companion page on five ways ethical reasoning characteristically goes wrong
- **Applied Dilemmas** — theory-agnostic cases where competing principles lead to genuinely different conclusions, accompanied by structured inquiry questions, stakeholder analyses, and writing prompts for seminar or written assignment use

The site is designed for undergraduate and graduate instruction but belongs to anyone who takes seriously the difficulty of living and deciding well.

## Technical Notes

The site is built entirely in hand-coded HTML with no frameworks, no JavaScript libraries, no build tools, and no external dependencies beyond Google Fonts. Each page is a fully self-contained HTML file with inline CSS. There is no server-side logic, no CMS, and no database.

This is a deliberate choice. It keeps the site fast, portable, and easy to adapt. Anyone who can open a text editor can contribute a page. The full site can be downloaded, mirrored, or forked without any installation or configuration.

The site is hosted on GitHub Pages and served at a custom domain via Namecheap DNS.

## Design System

All pages share a consistent visual identity:

- **Background:** `#f5f0e8` (parchment) with a subtle cartographic grid overlay
- **Dark header:** `#2c2416` with map grid
- **Accent:** `#b8860b` (antique gold)
- **Fonts:** EB Garamond (body) · Libre Baskerville (headings)
- **Motif:** cartographic — compasses, coordinates, uncharted territory

The design metaphor is intentional. Ethical frameworks are navigational instruments, not algorithms. The site's visual language reflects that.

## Contributing

Moral Latitude is at an early stage and welcomes fellow travellers. If you are an educator, graduate student, philosopher, or careful thinker who would like to propose a dilemma page, a framework page, or a correction, the best current path is to open a GitHub Issue describing your proposal, or to reach out directly.

Contributions should be consistent with the site's pedagogical philosophy: dilemmas should be genuinely open, frameworks should be presented with acknowledged limitations, and no page should imply that careful reasoning always converges on a single correct answer.
